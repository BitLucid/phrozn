Permalinks Strategies

Currently there's only one permalink strategy implemented.

However, Phrozn was designed in a way to allow any number of them - more info on this in successive releases.

Parametrized Permalink

Basically, you can specify explicitly where to output any given entry:

Permalink Setting Output File
/some/path/ /some/path/index.html
/some/path /some/path.html
/some/path/:var/
provided that var is set to "value"		 /some/path/value/index.html
/some/path/:var
provided that var is set to "value"		 /some/path/value.html

So, in order to output a given entry into /sub/folder/, all you need to do is to include permalink: /sub/folder/ variable into entry's YAML Front Matter: 

permalink: /sub/folder
---
<p>My first entry</p>
Example of using parameters in permalink: 

date: "2011-04-01"
permalink: /sub/folder/:date/
---
<p>My first entry</p>

Above entry will be written into /sub/folder/2011-04-01/index.html file.
